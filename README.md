# 全功能专业仓储与供应链管理系统（WMS+SCM）

> 基于 React + TypeScript + Tailwind CSS 的专业仓储管理系统原型

## 系统简介

这是一套功能完整的**仓储管理系统（WMS）+ 供应链管理系统（SCM）**模型，包含了从采购入库、销售出库、库内管理到财务结算的完整业务流程。

### 核心亮点

1. **极简拣货法**：通过"SKU聚合视图"解决拣货员"满场跑"的问题
2. **内部库位联**：解决"退货找不着位"的问题
3. **财务严谨性**：通过"差异处置流程"确保采购、仓库、财务三维度账目闭环
4. **招商赋能**：将"展厅"定义为特殊计费库位，具备仓储招商平台能力
5. **最后关卡拦截**：出库前扫码，将取消订单风险降至最低

## 功能模块

### 1. 基础信息管理
- 商品/SKU管理 - 维护商品图片、规格、SKU编码
- 库位/货架管理 - 数字孪生定义、分区管理、类型设置
- 供应商管理 - 供应商信息、结算周期、供货类目

### 2. 采购入库
- 采购单管理 - 采购需求、订单跟踪
- 入库作业 - PDA扫码入库、上架引导、库位推荐
- 差异处理 - 到货差异自动记录、实际成本计算

### 3. 销售出库
- 订单管理 - 多平台订单统一处理
- SKU聚合拣货 - 按SKU维度聚合订单，优化拣货路径
- 打包发货 - 组合面单打印、内部库位联、多品订单分流

### 4. 库内管理
- 库存监控 - 实时库存水位、虚拟库存与实物库存同步
- 移库/补货 - 存储区到拣货区调拨、PDA扫码移库
- 盘点管理 - 周期盘点、动态盘点、差异报告

### 5. 拦截与退货
- 出库拦截 - 出库前扫描面单，自动拦截已取消订单
- 退货处理 - 退货归位引导、异常件处理

### 6. 预警中心
- 库存预警 - 安全阈值设置、自动补货/滞销预警
- PDA设备监控 - 设备状态、实时作业动态

### 7. 财务结算
- 仓储计费 - 按存储时长、库位类型、件数自动核算
- 采购核算 - 订单利润分析、实际成本分批次核算
- 财务报表 - 营收报表、周转报表、采购差异报表

### 8. 数字化运营看板
- 运营总览 - 关键指标可视化
- 数字孪生 - 仓库可视化布局、实时状态
- 预警中心 - 多维度预警配置与明细

## 技术栈

- **前端框架**: React 18 + TypeScript
- **路由**: React Router DOM v6
- **状态管理**: TanStack Query
- **UI组件**: shadcn/ui + Radix UI
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **图表**: Recharts
- **构建工具**: Vite

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173 查看系统。

### 构建生产版本

```bash
npm run build
```
