**全功能专业仓储管理系统（WMS+SCM）模型清单**：
这套功能模型现在既包含了“展厅招商、SKU聚合拣货、面单关联库位联”**等实战创新点，也补齐了专业WMS系统必备的**“PDA作业、库内移库、财务结算”**等底层架构。
## 仓储管理系统（WMS）全功能模型清单

| 一级模块 | 二级模块 | 功能描述与关键业务细节（融合精华版） |
| :--- | :--- | :--- |
| **1. 基础信息管理** | **1.1 商品/SKU管理** | 维护商品图片、规格、SKU编码。支持厂家条码与系统码映射。 |
| | **1.2 库位/货架管理** | **数字孪生定义**：分区（存储区、拣货区、展位区）、库位编号（A-01-01）、类型设置（普通库位、展位）。 |
| | **1.3 供应商管理** | 维护供应商信息、联系方式、结算周期、供货类目负责人。 |
| | **1.4 仓库/多仓设置** | 配置仓库基本信息，支持物理仓库与虚拟仓库（展厅）的多仓协同管理。 |
| **2. 选品与招商中心** | **2.1 选品中心审核** | 内部商品提交后，由审核员审核规格、价格，通过后发布至选品池。 |
| | **2.2 招商入驻管理** | 供应商入驻申请；支持“继仓模式”或“代贴单模式”协议选择。 |
| | **2.3 展位租赁管理** | **核心创新**：支持按库位类型（展位）定义计费规则（按天、按面积），自动生成租金账单。 |
| | **2.4 自有会员商城** | 基于选品池，针对会员设定独立价格体系，支持等级折扣与分佣玩法。 |
| **3. 采购与入库管理** | **3.1 采购单/需求单** | 根据预警或订单缺货自动生成。无货源模式下抓取第三方（如1688）单号。 |
| | **3.2 分批入库差异处理** | **差异处置**：处理“到100实收90”等情况，支持异常挂起、采购员二次确认及退款流转。 |
| | **3.3 采购成本核算** | 记录实付金额。**算法**：总金额/实入库量 = 实际单件成本。支持分批次成本核算。 |
| | **3.4 上架引导(PDA)** | 系统根据优先级（A-B-C）推荐库位，PDA扫码核对，防止规格放错。 |
| **4. 销售与订单处理** | **4.1 多端订单抓取** | 对接淘宝、抖音、拼多多等。有现货自动预占库，无现货生成采购需求。 |
| | **4.2 待发货(SKU聚合)** | **核心视图**：以SKU维度聚合订单，展示总需求量、对应库位及优先级（备货波次）。 |
| | **4.3 定价策略管理** | 针对不同渠道（平台售价、会员价、供货价）设定独立的价格模型。 |
| **5. 出库作业(拣货打包)** | **5.1 智能波次拣货** | 拣货员根据SKU聚合视图，到指定库位一趟拿完所有所需商品。 |
| | **5.2 组合面单打印** | **关键设计**：同时打印平台面单+内部识别联。内部联含**库位编码**，两者物理关联贴标。 |
| | **5.3 边拣边贴/打包** | 在拣货区或操作台进行匹配打包。支持爆款“预打包”后直接贴单出库。 |
| | **5.4 多品订单分流** | 系统自动剔除多SKU复杂订单，交由专人或专门区域独立处理。 |
| **6. 拦截、退货与复核** | **6.1 出库扫描拦截** | **最后一道关卡**：出库前扫描面单，自动识别并拦截已取消订单。 |
| | **6.2 快速归位引导** | 拦截/退货包裹利用内部联上的库位信息，无需查系统，直接“回仓归位”。 |
| | **6.3 异常件处理** | 标记拦截异常、破损异常、漏发异常，流转至采购或客服跟进。 |
| **7. 库内管理与PDA** | **7.1 库存实时监控** | 实时跟踪各库位水位。支持虚拟库存（展厅）与实物库存双向同步。 |
| | **7.2 库内移库/补货** | 触发式补货：从存储区到拣货区/展位的调拨。支持PDA扫码移库。 |
| | **7.3 盘点与日志** | 支持周期性盘点、动态盘点。记录所有操作日志，确保责任到人。 |
| **8. 财务结算与计费** | **8.1 仓储计费管理** | 根据存储时长、库位类型、件数自动核算供应商/商户的仓储费。 |
| | **8.2 采购/订单核算** | 订单利润分析（售价 - 采购成本 - 物流 - 仓储费 - 损耗）。 |
| | **8.3 财务报表生成** | 营收报表、周转报表、采购差异报表、分中心盈利报表。 |
| **9. 看板与数字化监控** | **9.1 数字化运营看板** | 可视化展示当日发货量、拣货效能、库存水位、订单拦截量。 |
| | **9.2 数字孪生/平面图** | 仓库可视化布局，直观展示各库位商品分布状态，提升查找效率。 |
| | **9.3 库存预警中心** | 设定安全阈值（如5天备货期），自动发出补货/滞销预警。 |

---

### 系统设计的核心亮点总结：

1.  **极简拣货法**：通过**“SKU聚合视图”**解决拣货员“满场跑”的问题，通过**“内部库位联”**解决“退货找不着位”的问题。
2.  **财务严谨性**：通过**“差异处置流程”**和**“不可篡改的成本核算”**，确保了采购、仓库、财务三个维度的账目完全闭环。
3.  **招商赋能**：将**“展厅”**定义为特殊的计费库位，让系统具备了从单纯的“自营WMS”向“仓储招商平台”转型的商业能力。
4.  **最后关卡拦截**：在包裹“出门”前扫码，将所有前端取消订单的操作风险降至最低，避免物流费损失。
