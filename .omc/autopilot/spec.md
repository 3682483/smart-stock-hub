# WMS+SCM 仓储管理系统规格说明书

## 1. 项目概述

**项目名称**: 群兴仓储管理系统（WMS+SCM）
**项目类型**: Web管理后台原型系统
**核心功能**: 模拟完整仓储管理流程，包含采购入库、销售出库、库内管理、财务结算等38个功能点
**目标用户**: 仓储管理员、采购员、财务人员

## 2. 功能模块清单

| 功能 | 路由 | 组件 | 模拟数据 |

## 3. 技术规范

### 3.1 技术栈
- React 18 + TypeScript
- React Router DOM v6
- TanStack Query
- Lucide React (图标)
- Tailwind CSS + shadcn/ui组件

### 3.2 模拟数据规范
每个页面必须包含：
- 列表展示（表格/卡片）
- 新增/编辑功能（弹窗表单）
- 删除/批量操作
- 状态切换
- 分页或虚拟滚动
- 搜索/筛选

### 3.3 交互规范
- 所有按钮可点击
- 所有表单可提交
- 所有模态框可打开/关闭
- Toast提示反馈
- 加载状态展示

## 4. 页面详细设计

### 4.1 DashboardPage (运营看板)
```
- 关键指标卡片: 今日发货量、待拣货、库存预警、采购待入库
- 图表: 发货趋势图、库存水位图
- 待办列表: 预警事项、待处理订单
```

### 4.2 ProductsPage (商品/SKU管理)
```
- 商品表格: 图片、SKU、名称、规格、库存、状态
- 操作: 新增商品、编辑、删除、上下架
- 筛选: 分类、状态、供应商
```

### 4.3 LocationsPage (库位管理)
```
- 平面图视图: 仓库分区可视化
- 库位表格: 编号、类型(A区/B区/展位)、容量、状态
- 操作: 新增库位、编辑、设置类型
```

### 4.4 SuppliersPage (供应商管理)
```
- 供应商卡片: 名称、联系人、结算周期、供货类目
- 操作: 新增供应商、编辑、查看详情
```

### 4.5 PurchaseOrdersPage (采购单管理)
```
- 采购单表格: 单号、供应商、商品、数量、金额、状态
- 操作: 新建采购单、确认入库、取消
- 状态: 待确认、部分入库、已完成
```

### 4.6 InboundPage (入库作业)
```
- 入库单列表: 到货单号、供应商、实收数量
- PDA上架引导: 推荐库位、扫码确认
- 差异处理: 异常挂起、二次确认
```

### 4.7 SalesOrdersPage (订单管理)
```
- 多平台订单: 淘宝/抖音/拼多多
- 订单表格: 平台、单号、商品、收件人、状态
- 操作: 批量处理、标记发货、取消
```

### 4.8 PickingPage (SKU聚合拣货)
```
- SKU聚合视图: 聚合后SKU、总需求、库位
- 波次信息: 优先级、拣货顺序
- 操作: 开始拣货、完成波次
```

### 4.9 PackingPage (打包发货)
```
- 待打包列表: 订单、拣货完成时间
- 面单打印: 平台面单+内部识别联
- 操作: 确认打包、打印面单、完成发货
```

### 4.10 InventoryPage (库存监控)
```
- 库存列表: SKU、库位、可用数量、锁定数量
- 水位监控: 实时库存状态
- 操作: 库存调整、库位变更
```

### 4.11 FinancePage (财务结算)
```
- 财务报表: 营收、成本、利润
- 仓储费核算: 按库位、时长、件数
- 采购核算: 实际成本、分批次
```

### 4.12 新增页面: TransferPage (移库/补货)
```
- 调拨单列表: 源库位、目标库位、商品、数量
- PDA扫码: 移库确认
- 操作: 新建调拨、确认完成
```

### 4.13 新增页面: StocktakingPage (盘点管理)
```
- 盘点任务: 任务编号、类型(周期/动态)、状态
- 盘点明细: 商品、账面数、实盘数、差异
- 操作: 新建任务、录入实盘、确认差异
```

### 4.14 新增页面: InterceptPage (出库拦截)
```
- 拦截记录: 扫描时间、面单号、订单状态、拦截原因
- 操作: 扫描面单、确认放行、异常处理
```

### 4.15 新增页面: ReturnsPage (退货处理)
```
- 退货单列表: 退单号、原订单、原因、状态
- 归位引导: 内部联库位信息
- 操作: 确认收货、归位完成、异常处理
```
